# Self-hosted fonts

These WOFF2 files are self-hosted so the archetype finder has zero external
network dependencies (no CDN, no Google Fonts call).

- **Inter** (`inter-*.woff2`) — UI text. SIL Open Font License 1.1.
  https://github.com/rsms/inter
- **IBM Plex Mono** (`plexmono-*.woff2`) — scores, labels, datelines.
  SIL Open Font License 1.1.
  https://github.com/IBM/plex

Both are OFL-licensed and free to embed and redistribute. Latin subset only.
